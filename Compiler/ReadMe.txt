========================================================================
    控制台应用程序：Compiler 项目概述
========================================================================

简介
	该项目是一个简单的C语言编译器，支持int、char两种类型，支持if-else语句、for语句、while语句，支持变量和数组的申明、表达式的计算、逻辑运算。


Compiler.vcxproj
    这是使用应用程序向导生成的 VC++ 项目的主项目文件，其中包含生成该文件的 Visual C++ 的版本信息，以及有关使用应用程序向导选择的平台、配置和项目功能的信息。

Compiler.vcxproj.filters
    这是使用“应用程序向导”生成的 VC++ 项目筛选器文件。它包含有关项目文件与筛选器之间的关联信息。在 IDE 中，通过这种关联，在特定节点下以分组形式显示具有相似扩展名的文件。例如，“.cpp”文件与“源文件”筛选器关联。

Compiler.cpp
    这是主应用程序源文件。

/////////////////////////////////////////////////////////////////////////////
头文件:

declaration.h
	类及函数的申明。

macro.h
	不同类型字符的宏定义。

structure.h
	程序中使用到的数据结构的定义。

/////////////////////////////////////////////////////////////////////////////

cpp文件:

Compiler.cpp
	main函数所在文件，控制编译器工作流程。

Function.cpp
	常用函数的实现。

Token_scanner.cpp
	词法分析器类的成员函数的实现。

/////////////////////////////////////////////////////////////////////////////
其他标准文件:

StdAfx.h, StdAfx.cpp
    这些文件用于生成名为 Compiler.pch 的预编译头 (PCH) 文件和名为 StdAfx.obj 的预编译类型文件。
